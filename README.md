# Dicee Game 🎲

A simple dice game built using HTML, CSS, and JavaScript.

## Features
- Random dice roll generation
- Dynamic dice image updates
- Winner announcement for Player 1 or Player 2
- Draw detection

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Open the game in a browser.
2. Refresh the page.
3. The dice are rolled automatically.
4. The player with the higher dice value wins.
